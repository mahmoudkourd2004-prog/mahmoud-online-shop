# 🛒 Mahmoud Online Shop

A modern and responsive e-commerce website built using HTML, CSS, and JavaScript.
This project features dynamic product sliders, shopping cart functionality, and a clean UI design.

---

## ✨ Features

* 🛍️ Dynamic product sections
* 🛒 Shopping cart system
* 🎠 Swiper.js product sliders
* 🔍 Search & category filtering UI
* 📱 Fully responsive design
* 💳 Checkout page

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript
* Swiper.js
* Font Awesome
